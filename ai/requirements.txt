# ────────────────────────────────────────────────────────────────
# PDF EXTRACTION — three libraries, tried in order
# (extract_pdf.py uses all three as a fallback chain)
# ────────────────────────────────────────────────────────────────

# Primary PDF extractor — best quality for complex layouts,
# multi-column text, and embedded tables.
# Used by: extract_pdf.py  (method 1)
pdfplumber>=0.11.0

# Fallback PDF extractor — lightweight, good for simple PDFs.
# Note: PyPDF2 was superseded by pypdf in v3+. Either works;
#       extract_pdf.py imports whichever is available.
# Used by: extract_pdf.py  (method 2)
pypdf>=4.0.0

# Deep fallback PDF extractor — handles edge-case PDFs that
# pdfplumber and pypdf fail on. Slower but more thorough.
# Used by: extract_pdf.py  (method 3)
pdfminer.six>=20231228
